# 🧠 NeuroHelper

Generate plans, checklists, templates, and ideas in seconds.

## 🚀 Launch (5 sec)

```bash
docker compose up -d --build
```

✅ http://localhost:8080

## ✨ Features

- 📅 **Day plans** with time
- ✅ **Task checklists**
- 📝 **Text templates**
- 💡 **Ideas** for work/study
- Real-time response streaming

## On the server

```bash
PUBLIC_PORT=80  # in .env
docker compose up -d --build
```
